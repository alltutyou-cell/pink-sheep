# 💕 Valentine's Day Sheep Website

A fun, interactive Valentine's Day website featuring adorable sheep asking "Will You Be My Valentine?"

## 🎨 Features

### Visual Design
- **Vibrant Pink Gradient Background** - Animated gradient that shifts smoothly
- **Floating Hearts** - 10 animated hearts floating up the screen continuously
- **Bouncing Sheep** - 6 cute sheep positioned around the page, gently bouncing
- **Glassmorphism Card** - Beautiful frosted glass effect for the main content
- **Sparkles** - Twinkling stars scattered across the page

### Interactive Elements
- **Runaway "NO" Button** - The button actively dodges your mouse cursor immediately! It shakes and jumps to a new spot if you try to click it.
- **"YES!" Button** - Growing bigger with each failed attempt to click "No", eventually taking over the screen.
- **Sheep Easter Egg** - Click on any sheep to hear a visual "Baa!" 🐑 and see them bounce.
- **Playful Messages** - If you manage to catch the "No" button (it's hard!), you'll see guilt-tripping messages like:
  - "The sheep are crying now... 😭"
  - "They bought you flowers! 🌹"

### Celebration Animation
When "YES!" is clicked:
- 🎉 Confetti explosion with pink hearts
- 💕 Happy message: "YAY! The sheep are so happy! 💕🐑"
- 🐑 All sheep jump for joy
- ✨ Screen brightness pulse effect

## 🚀 How to Run

1. **Start a local server:**
   ```bash
   python3 -m http.server 8000
   ```

2. **Open in browser:**
   Navigate to `http://localhost:8000`

## 📸 Adding Your Own Photos

Want to see your special someone floating around? It's easy!

1.  **Open the project folder:** Go to `/Users/macair/.gemini/antigravity/playground/nascent-trifid/`
2.  **Open the `photos` folder:** You'll see some placeholder sheep images there.
3.  **Add your photos:**
    *   Drag and drop your photos into this folder.
    *   **Rename them** to numbers: `1.jpg`, `2.jpg`, `3.jpg`, etc.
    *   The website is set to look for up to **10 photos** by default.
    *   You can replace the existing sheep photos with your own!

The photos will randomly appear floating up from the bottom of the screen like cute memories! 📸💕

This website follows modern web design principles:
- **Premium Aesthetics** - Vibrant colors, smooth gradients, and glassmorphism
- **Micro-animations** - Subtle movements that bring the page to life
- **Interactive Engagement** - Playful button behavior encourages interaction
- **Responsive Design** - Works beautifully on all screen sizes
